Web Development Project: HTML

Use HTML language to construct a webpage introducing somebody or something (e.g., yourself, your favorite movie character, a cool electronic gadget …)

  • The webpage should include at least the following elements: two images, one hypertext link, one character entity, one HTML comment, one 2-level nested ordered list, and one table with no less than two rows.
  • The two images files should be put in a directory named "images". The "images" directory is in the same directory as the webpage HTML file.
  • The webpage should also contain an additional HTML tag of your choice, not mentioned above.
  • The webpage should pass HTML5 validation.

8. (5 points) What is the difference between the inline element and block element? List one inline element and one block element appearing in the HTML source code of the webpage.

 

Block elements contain an entire large region of content. For example, <p>.
Inline elements affect a small amount of content. For example, <img>.
